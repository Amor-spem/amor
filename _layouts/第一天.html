HTML元素

        html中元素指的是开始标签到结束标签的所有代码
        标签中间的内容也就是元素内容
        大部分元素可以拥有属性
        大多数 HTML 元素可以嵌套
        HTML 标签对大小写不敏感,但推荐小写

HTML属性

        属性提供了有关HTML元素的更多信息
        属性以名称/值的形式出现,例如:name="value"
        属性出现在开始标签中,例如a标签元素中的开始标签定义了超文本链接的地址
        属性加引号,单双均可,错开使用可使值本身含有双引号
        属性              值                  含义
        class             classname           元素的类名
        style             style_definition    元素行内样式
        id                id                  元素唯一id
        title             text                元素的额外信息

HTML标题

        h1到h6定义标题 1最大 6最小 浏览器会在标题后方自动添加空行
        搜索引擎使用标题为的网页的结构和内容编制索引,对用户也是
        所以仅仅把标题标签用于标题文本。不要仅仅为了产生粗体文本而使用它们

HTML段落

        p定义段落
        会自动在段落后添加空行(p)是块级元素
        HTML中的连续空行或者空格都会被算作一个空格

HTML样式

        外部样式表 当样式要运用在多个页面时 引入外部样式表减少代码量
        内部样式表 在head位置的style样式
        内联样式表 在个别元素需要的样式时,直接在相关标签中使用样式属性
        style可以改变HTML的样式
        避免使用 center font basefont s strike u align bgcolor color 使用样式来代替
        background-color:blue 将当前元素背景修改为这个色
        style淘汰了旧的align的属性
