初识脚本JavaScript

script 标签用于自定义脚本
可以是包含脚本内容,也可指向外部地址
type属性是必须的,规定了脚本的mime类型
主要用于 图片操作 表单验证 内容动态更新等

其中可以使用noscript标签作为替代内容,在脚本无法生效时会显示替代内容

文件路径

相对路径以当前所在文件夹为基本点,可以直接指向下层路径
也可以用../的方式指向上层路径(最好使用相对路径,以保证在不同的环境下都能正常显示网页)

再探head标签

head元素是所有头部元素的容器 以下标签都可以添加到head标签内
title base meta script style

[title]

  • 定义浏览器工具中的标题
  • 页面被收藏时显示的标题
  • 搜索引擎结果中的标题

[base]

规定了默认地址 或默认目标,只能存在一个

[link]

定义了文档和外部资源之间的关系
最常用于链接样式表

[style]

[meta]

提供了文档的元数据,不会显示在界面,但对机器是可读的
规定页面的描述,关键词,文档的作者,最后修改时间等
始终位于head标签中

HTML实体 将预留字符打印出来

            scheme://host.domain:port/path/filename
        
    
            
  • scheme - 定义因特网服务的类型。常见的类型 http https ftp file
    • 
            
  • host - 定义域主机（http 的默认主机是 www）
    • 
            
  • domain - 定义因特网域名，比如 w3school.com.cn
    • 
            
  • :port - 定义主机上的端口号（http 的默认端口号是 80）
    • 
            
  • path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
    • 
            
  • filename - 定义文档/资源的名称
